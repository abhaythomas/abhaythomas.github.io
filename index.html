Project 04 · LLM Fine-tuning · Research
LLaMA Fine-tuning
for Abstract Reasoning
LoRA · PEFT · ARC Challenge · Kaggle

Fine-tuned LLaMA-3.1-8B using LoRA (PEFT) on the ARC Challenge benchmark — one of the hardest tests of abstract reasoning in AI. Curated and preprocessed training data with human reasoning annotations, engineered structured prompts, and ran a grid search across 54 hyperparameter configurations. Explored candidate sampling with a BigBird-RoBERTa ranking model to improve robustness. Key finding: explicit reasoning annotations significantly outperform raw fine-tuning alone.
